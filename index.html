About

Details of our steps from start to finish!

  • Step 1

    Data Set: Kaggle

    CSV file of historical UFC-Fight data from 1993 to 2021. Data contains a compilation of info about each fighter and fights.

  • Step 2

    Data Optimization

    ETL process included data clean up, web scraping, bringing in latest fight stats, appending / removing irrelevent columns, reformating data for model creation.

  • Step 3

    Data Modeling

    A test train model was used to classify previous fight data stats in order to predict probablistic outcomes.

  • Step 4

    Execution

    Through the use of Atlas Mongo and Heroku, we were able to create a server to visual the website and databases for our viewers.
